# MindBridge

  A premium mental health counseling platform for school counselors, consisting of a web portal and a mobile app.

  ---

  ## 📁 Repository Structure

  ```
  /                  — Counselor Portal (web app)
  mobile/            — Counselor Mobile App (Expo / React Native)
  ```

  ---

  ## 🖥️ Counselor Portal (Web)

  A feature-rich web portal built with modern frontend technologies.

  **Tech Stack:**
  - **React + Vite** — fast modern frontend
  - **Tailwind CSS v4** — utility-first styling
  - **Framer Motion** — smooth animations
  - **shadcn/ui** — polished component library
  - **Recharts** — data visualization
  - **Wouter** — lightweight routing

  **Features:**
  - Counselor onboarding flow
  - Student management
  - School visit tracking
  - Revenue & reports dashboards
  - Settings with profile management

  ---

  ## 📱 Counselor Mobile App (Expo)

  A native mobile app for counselors on the go, built with Expo and React Native.

  **Tech Stack:**
  - **Expo (SDK 52)** — React Native framework
  - **Expo Router** — file-based navigation
  - **React Native Reanimated** — smooth animations
  - **TypeScript** — type-safe codebase

  **Features:**
  - Authentication flow with OTP verification
  - Onboarding experience
  - Dashboard with revenue overview and analytics
  - Appointments management
  - Student roster with detailed profiles
  - School visit tracking
  - Dark/light mode support
  - Settings screen

  **Location:** `mobile/` directory

  ---

  ## Getting Started

  ### Web Portal
  ```bash
  npm install
  npm run dev
  ```

  ### Mobile App
  ```bash
  cd mobile
  npm install
  npx expo start
  ```
