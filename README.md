# MindBridge Counselor Portal

  A premium mental health counseling platform portal built with:
  - **React + Vite** — fast modern frontend
  - **Tailwind CSS v4** — utility-first styling
  - **Framer Motion** — smooth animations
  - **shadcn/ui** — polished component library
  - **Recharts** — data visualization
  - **Wouter** — lightweight routing

  ## Features
  - Counselor onboarding flow
  - Student management
  - School visit tracking
  - Revenue & reports dashboards
  - Settings with profile, specialization, security, notifications, and more

  ## Getting Started
  ```bash
  pnpm install
  pnpm --filter @workspace/counselor-portal run dev
  ```
